# Security Policy

## Supported Versions

The following versions of Tidy App are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow the steps below:

1. **Contact**: Send a detailed description of the vulnerability to [security@tidyapp.com](mailto:security@tidyapp.com). Please do not use this email address for any other reasons.

2. **Disclosure**: Once we receive the report, we will work to analyze, confirm, and reproduce the issue. We typically respond within one week.

3. **Remediation**: After verifying the vulnerability, we will work on a fix, which will be released as soon as possible, typically within 90 days.

4. **Acknowledgment**: We value the contributions of security researchers and will publicly acknowledge your contribution once the vulnerability is resolved, unless you wish to remain anonymous.

Thank you for helping to keep Tidy App secure.
